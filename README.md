# LightDoc Reader

LightDoc Reader is a mobile-friendly `.lpdx` reading app built for viewing Light Progressive Document Exchange files.

## Features

- Open `.lpdx` files from local device storage
- Drag and drop support on desktop
- Android-responsive layout
- Reading stats, metadata, and structured blocks
- Recent files list saved locally in the browser
- Static export ready for Vercel and other static hosts

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The project uses static export, so it can be deployed easily on Vercel.
